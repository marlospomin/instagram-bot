# Instagram Bot

> A simple instapy script to grind Instagram followers.

## Usage

Install the dependencies using the script found in the `setup` folder and you're set.

`python3 instagram.py --user username --password password`

## Requirements

* Chrome 71 or above.
* Python3.
* Instapy.
